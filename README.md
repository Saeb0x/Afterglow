# Afterglow Engine
Afterglow is a cross-platform, high performance game engine currently in development. This readme will be updated with more info eventually.

## Features

- [x] **OpenGL Renderer featuring advanced techniques like texture batching for optimized rendering performance.**
- [x] **Scene Manager.**
- [x] **Entity-Component-System (ECS) Framework.**
- [x] **Level Serialization/Deserialization for the loading and saving of scenes.**
- [x] **Resource Pool for efficient management of frequently used resources like textures and shaders.**
- [x] **Initial implementation of the Level editor featuring UI elements, including a Content Browser powered by ImGui.**
